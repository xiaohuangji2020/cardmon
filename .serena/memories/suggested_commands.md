# Suggested Commands for Cardmon Development

## Godot Editor
- Open project: `godot` (from project root)
- Run project: F5 in Godot editor or `godot --play` from CLI

## MCP Server Management
- MCP Server runs on port 3571
- Runtime Server runs on port 3572
- Configuration file: `godot_mcp_config.json`

## Git Commands
- Check status: `git status`
- View recent commits: `git log --oneline -n 10`
- Current branch: master

## File Operations
- List directory: `ls -la`
- Find files: `find . -name "*.gd"` (for GDScript files)
- Search in files: `grep -r "pattern" --include="*.gd"`

## System Info
- Platform: macOS (Darwin 22.6.0)
- Working directory: /Volumes/MacintoshData/mine/cardmon
